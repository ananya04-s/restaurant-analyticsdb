# Restaurant Analytics Dashboard (Power BI)

## Overview
This project is an interactive Power BI dashboard designed to analyze restaurant customer and billing data. It provides insights into customer demographics, total bills, smoking preferences, and day-wise trends through visually appealing reports.

## Features
- 📈 KPI cards for male and female customer counts
- 💰 Total bill analysis by gender
- 🚬 Smoking preference statistics
- 📅 Day-wise customer distribution
- 🎨 Interactive and visually rich dashboard design
- ⚡ Dynamic filtering and data exploration

## Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Excel/CSV Dataset

## Key Insights
- Compare male and female customer distribution.
- Analyze billing patterns across demographics.
- Explore smoking preferences among customers.
- Visualize customer trends by day of the week.
